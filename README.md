# OpenAEM
An wall turbulence simulator powered by attached eddy model
## Usage
1. setup an virtual environment
    ```console
    python3 -m venv .venv
    source .venv/bin/activate
    ```
2. install locally for development
   ```console
   python -m pip install -e .
   ```
